Made by employees, for employees

Profiles helps you keep your company profile updated and integrates with the tools you already use. We've built Profiles on PHP and Angular with the feedback from our awesome colleagues.

Active Directory

You can log in and manage your profile by using your Active Directory credentials - no new accounts, no forgotten passwords, everything is kept in sync with your everyday tools.

Open Feedback

We've chosen to make the feedback process open for the entire group. Simply visit the issues section on GitHub and let us know what we can do better for you.

Open Collaboration

We've built Profiles on open-source technologies and tools so that everyone can collaborate - Angular (frontend), PHP (backend), Travis (continuous integration), Scrutinizer (code quality).
